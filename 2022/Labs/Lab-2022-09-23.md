# Lab-2022-09-23

Questions goes simpler to harder.

1. Please write a code which ask the user for their percentage grade and changes this grade to [American letter grade](https://en.wikipedia.org/wiki/Academic_grading_in_the_United_States)

2. Please write a code which calculates [Modified Bavarian formula](https://en.wikipedia.org/wiki/Academic_grading_in_Germany).
You need to ask minimum possible grade, maximum possible grade and actual grade from the user.


3. Please write a program which find the greatest number among three numbers.
You will ask the user for these three numbers and print greatest of them in the screen.
