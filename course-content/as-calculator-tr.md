# hesap makinesi olarak Python 


Python hesap makinesi olarak kullanılabilir.
Python, okuldan bildiğiniz gibi gerekli tüm operatörlere ve kurallara sahiptir.


```python
print(4+5)
print(4*5)
print(5/2)
print(5%2)
print(5//2)
print(2+1/3*3)
print((2+(1/3))*3)
print(2**5)
```

Ortak operatörler

- a + b toplama
- a - b çıkarma
- a * b çarpma
- a / b bölme (sonuç ondalık sayı)


Sadece aşağıdaki operatörler normal matematiksel operatörlerden biraz farklıdır.

- % kalan operatörüdür. [Kalan operatörü](useful-python-code-snippets-en.md#divisible-by-some-number), bir sayının başka bir sayıya bölünüp bölünemeyeceğini bulmak için birçok soruda kullanılır.

- // is tamsayı bölme.
- ** üs operatörü. 2\*\*5, 2\*2\*2\*2\*2'ye eşittir
