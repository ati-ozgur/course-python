## Lists


Strings are defined using []


    empty_list = []
    list_of_numbers = [1,3,5]


Adding new members to list

    new_list = [1,3]
    new_list = new_list + [2,4]
    print(new_list)
    # Or
    a_list = [1,3]
    a_list.append(2)
    a_list.append(4)


Indexing list is done via [index]

    new_list[0]
    # negative indexing start from end
    new_list[-1]


length of list is found using **len** function


    print(len(new_list))
